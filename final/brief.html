Battleship Game - Project Summary

Project Summary

For my final project, I plan to improve upon the visual design of my Battleship game.

Project Description

For my Battleship game iteration 1, most of my time and energy went towards working out the game mechanics, so while I think it looks okay, I still spent a majority of my time on the code. I want to implement some additional CSS and JavaScript, and maybe some animations so that it is more visually appealing. Currently, the display is mostly text-based, but I want to also find a way to incorporate images for the player's ships. As well, I think improving the overall visual aesthetics will also contribute to the game being easier to play and increase usability.

Expert Feedback
